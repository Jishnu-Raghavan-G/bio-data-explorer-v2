\# Disease vs Healthy Data Explorer



\## What is this?

This project compares gene expression between healthy and disease conditions using Python.



\## What it does

\- Filters dataset into healthy and disease groups

\- Computes average expression

\- Visualizes comparison using bar chart



\## Why I made this

To understand how biological data can be analyzed using computational methods.



\## Future improvements

\- Add gene-wise comparison

\- Build interactive interface

\- Use real biological datasets
